常见问题

更新时间：2020年9月11日

  1. 文献管理软件有很多，英文有Endnote，Mendeley，Zotero，中文支持较好的的有Noteexpress，各有优缺点，这篇文章仅介绍Endnote。如果看英文较多或不用校园网系统，个人也超级推荐Zotero（开源，扩展性好）。
  2. Endnote试用和免费问题：Endnote本身是需要序列号注册的，给大家分享的版本是校园网内下载的，我使用过程中没有试用的问题。（1）高校学生，一般学校会购买相关资源；（2）如果出现试用到期问题，请大家前往下载别人提供的破解版。相关网址如下：https://www.jb51.net/softs/637860.html，https://www.isharepc.com/9496.html。（3）如果仍然不行，但还想要使用Endnote，可以尝试淘宝买个正版；（4）考虑直接放弃Endnote，转而使用Zotero。
  3. Endnote中文文献导入大多数无法识别文献信息，建议参考文章中提到的，先在Google Scholar或知网或百度学术等导出文献条目信息，再把文献以附件形式添加到软件中，个人感觉更方便。
  4. 使用【3】的文献导入方法，如果想要批量导入，请把文献条目信息（后缀可以是txt，enl等）和文献（例如pdf）设置为同一名字，放在同一文件夹下。选择Import folder，可以实现批量导入。

本期介绍Endnote的简易使用教程

1、Endnote是什么？

Endnote是一款被广泛使用的文献管理软件，其是SCI（Thomson Scientific 公司）的官方软件，支持国际期刊的参考文献格式有3776 种【也可以自定义期刊引用格式】。

软件非常方便科研狗进行文献整理，写笔记，做备注，分类，导出期刊格式，进行数据迁移【这个特别方便，可以随时将其他电脑上的文献导入到自己的电脑中】

2、文献管理软件这么多，为什么要用Endnote？

其实文献管理，你什么软件都不用都是可以的…但是对于需要大量看文献，需要对文献进行分类等功能时，Endnote无疑是最好用的之一。Endnote对英文文献的支持比较好，中文稍差，其他的文献管理软件像NoteExpress，我也用过，但是感觉上仍然觉得Endnote比较好。

3、从哪里获取？

其实各大高校应该都有购买相关软件，如果实在没有在校内数据库中找到，可以去北邮人等Ipv6网站资源上看看，也可以下载破解版的使用，基本没有差距。

02 新建数据库

第一次打开Endnote你需要建立一个库，相当于一个属于你自己的图书馆，用来放文献。点击File-New，会弹出框，这里提醒你要建立的库名称和保存位置（示例-Papers）。然后保存位置会出现2个文件，第一个是库存，里面有你的文献原件，第二个是目录，相当于一份清单。

04 文献导入

导入文献操作1：

  • 英文文献：从Google Scholar下载文献信息文件（文件后缀.enw）；科学上网或谷歌镜像即可
  • 中文文献：知网导出文献信息txt即可

导入文献操作2：从校内数据库或其他地方下载pdf（Xplore，elsevier）

导入文献操作3：导入到Endnote中

提醒大家，endnote导入文献的方法补充：

  1. 单独导入：操作在上文已提到，在获得一个文献的enw或txt之后导入，再在endnote附件区添加pdf；
  2. 成批导入：成批导入有两种，一种是大家可以下载一堆enw或txt文件导入文献信息后，在线查询文献下载。但这种方法校外网络等都不能实现。另一种是将文献pdf以及文献信息enw或txt文件，文件名命名相同后全部放在一个文件夹中，直接在endnote里【File】-【Import】-【Import folder】即可，endnote会自动匹配。

05 文献分组

对文献进行分组，单个文献可以在多个组内

06 文献笔记

导入文献文献笔记：针对某个文献可以建立自己的笔记

•创建某个word文件（其他文件，例如txt也可以），直接拖动到文献信息区【Reference】的【File Attachments】即可

•也可以直接记录在【Note】或者【Research Notes】板块

07 文献库迁移

【本地迁移】

•利用压缩功能把当前文献库保存，如果需要迁移到另一个电脑，可以直接拷贝该文件（文件类型.enlx）

选择上方菜单栏中的【File】-【Compress Library.enlx】，出现下方窗口

这里【注意！要选择With File Attachmnts，才能把你文献的信息和PDF等附件都进行压缩】，点击【Next】，就可以生成一个后缀为【.enlx】的文件。

•到另一个电脑上选择【Open Library】，导入该数据库即可【只需要导入.enlx后缀这一个文件，就可以实现全库转移】

【账号迁移】

可以登录Endnote账号，同步进行更新，但是需要科学上网，且同步更新效率较慢。建议本地迁移就好。
